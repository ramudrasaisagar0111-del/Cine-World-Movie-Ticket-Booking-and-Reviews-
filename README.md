# CineWorld – Movie Ticket Booking Website

## About The Project
CineWorld is a modern movie ticket booking website developed using HTML, CSS, and JavaScript.  
The website provides users with an interactive platform to explore movies, view details, book tickets, and post reviews.

The project is designed with a cinematic dark theme UI and focuses on responsive design, smooth navigation, and interactive user experience.

---

# Features

##  Home Page
- Displays currently running movies
- Movie cards with posters, ratings, and genres
- Movie filtering by category
- Responsive movie grid layout

##  Movie Details Page
- Detailed movie descriptions
- IMDb ratings
- Cast information
- Show timings
- Quick booking access

##  Booking System
- Dynamic seat selection
- Diamond & Gold seat categories
- Real-time booking summary
- Payment method selection
- Booking confirmation popup

##  Reviews Page
- Star rating system
- User review submission
- Audience reviews display
- Booking verification before review posting

##  Contact Page
- Email support
- Contact details
- Customer support section
